## Setup Instruction

### Using `requirements.txt`
- `pip install -r requirements.txt`
- `python app.py`


### Using `Pipfile`
- `pip install pipenv`
- `pipenv install`
- `pipenv run python app.py`


### Using Poetry `pyproject.toml`
- `pip install poetry`
- `poetry install`
- `poetry run python app.py`
